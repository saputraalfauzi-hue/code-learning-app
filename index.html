3. Membuat List di HTML

HTML Pemula 20 menit

Tujuan Latihan

Buat tiga jenis list di HTML: unordered list, ordered list, dan description list.

Persyaratan

Instruksi

Edit kode HTML di editor untuk membuat halaman dengan tiga jenis list:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ol>
  <li>Langkah pertama</li>
  <li>Langkah kedua</li>
  <li>Langkah ketiga</li>
</ol>

Pengujian

Setelah selesai, klik "Test Code" untuk menguji solusi Anda.

Kode belum diuji
Siap untuk mengetik kode...

Live Preview

Console

[10:30:45] Console siap. Kode Anda akan dieksekusi di sini.

Petunjuk

Gunakan tag <ul> untuk unordered list, <ol> untuk ordered list, dan <dl> untuk description list.

Setiap item dalam list menggunakan tag <li>.

Untuk description list, gunakan <dt> untuk term dan <dd> untuk deskripsi.

Solusi Contoh

<ul class="shopping-list">
  <li>Apel</li>
  <li>Pisang</li>
  <li>Jeruk</li>
</ul>

<ol class="steps-list">
  <li>Panaskan oven</li>
  <li>Campur bahan</li>
  <li>Panggang selama 30 menit</li>
</ol>

<dl class="definition-list">
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
