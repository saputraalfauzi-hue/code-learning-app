# CodeMaster - Platform Belajar HTML, CSS & JavaScript

Platform pembelajaran interaktif untuk belajar HTML, CSS, dan JavaScript dengan sistem latihan bertingkat.

## Fitur Utama

- **Editor Kode Lengkap**: Dengan syntax highlighting dan line numbers
- **Live Preview**: Tampilan real-time dengan mode desktop, tablet, dan mobile
- **Console Integrated**: Menampilkan output JavaScript
- **Sistem Latihan**: Latihan bertingkat dengan berbagai tingkat kesulitan
- **Progress Tracking**: Pelacakan progress pembelajaran
- **Hint System**: Petunjuk dan solusi untuk setiap latihan
- **Responsive Design**: Berfungsi optimal di semua perangkat
