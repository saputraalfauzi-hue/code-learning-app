# Aplikasi Belajar Website

Aplikasi pembelajaran web interaktif dengan fitur latihan, syllabus, chat AI, referensi, dan pelacakan progres.

## Fitur

1. **Dashboard**: Ringkasan aktivitas dan progres belajar.
2. **Latihan**: Soal latihan dengan berbagai tingkat kesulitan (mudah, sedang, sulit).
3. **Syllabus**: Rencana pembelajaran dengan topik terstruktur.
4. **Chat AI**: Asisten AI untuk menjawab pertanyaan tentang pemrograman web (menggunakan Gemini API).
5. **Referensi**: Kumpulan sumber belajar untuk HTML, CSS, JavaScript, dan tools.
6. **Progres**: Pelacakan dan visualisasi perkembangan belajar.

## Teknologi

- HTML5, CSS3, JavaScript (ES6+)
- Chart.js untuk visualisasi data
- Marked.js untuk rendering markdown
- Gemini API untuk fitur chat AI
- Local Storage untuk penyimpanan data lokal
